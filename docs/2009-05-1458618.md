# 数据挖掘之 DIG 
> 原文发表于 2009-05-17, 地址: http://www.cnblogs.com/chenxizhang/archive/2009/05/17/1458618.html 


所谓数据挖掘的DIG，是指微软将数据挖掘这种本来想当复杂的功能集成到了Office系统中（Excel和Visio）

 * Define Data
* Identify Task
* Get Result
