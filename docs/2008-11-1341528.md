# VS2008的一个更新补丁 
> 原文发表于 2008-11-26, 地址: http://www.cnblogs.com/chenxizhang/archive/2008/11/26/1341528.html 


之前安装VS2008的SP1之后有些莫名其妙的问题，例如智能感知全部变成了英文的，还有源代码调试也有问题。我为此专门找了微软的技术支持。 今天得到回复，微软新近推出了一个更新补丁，安装之后就可以解决了。鼓掌 Hello All, We finally completed the Loc Intellisense Hotfix that replaces the unlocalized Intellisense released in the SP1 loc versions. Please forward this information to anyone you think might need or want to know about this Hotfix. I’ll send a more complete status update tomorrow, but thought you would want to know where to send customers for the hotfix: KB Article: <http://support.microsoft.com/kb/957507> Download location and more information: <http://code.msdn.microsoft.com/KB957507> Also: Connect: <http://connect.microsoft.com/VisualStudio/Downloads/DownloadDetails.aspx?DownloadID=15258> Let me know if you find any problems. Please let me know what distribution lists we should send this information to! I’ll send a  more detailed Status email and update the project website tomorrow. Special Thanks to the large number of people who made this possible.   关于源代码调试，请参考下面这个链接 <http://blogs.msdn.com/sburke/archive/2008/01/16/configuring-visual-studio-to-debug-net-framework-source-code.aspx>
