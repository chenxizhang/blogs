# 谈一谈职业素养 
> 原文发表于 2012-04-08, 地址: http://www.cnblogs.com/chenxizhang/archive/2012/04/08/2437279.html 


内容摘要
----

 这一讲中，我分享了我对于职业素养的认识和理解。职业素养是一个很大的话题，但总的来说，他表现为某个人对于职业的一种精神和态度。

 我个人总结起来，职业素养是在具备专业技能基础上，表现出来的以下精神品质

 1. 责任心
2. 伦理观
3. 纪律性
4. 守时间
5. 乐沟通
6. 敢承担
