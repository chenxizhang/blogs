# 学习和自我学习 
> 原文发表于 2012-03-10, 地址: http://www.cnblogs.com/chenxizhang/archive/2012/03/10/2388593.html 


内容摘要
----

 学习是一种态度，也是一个有方法可循的过程，这一讲我分享一些我对于学习和自我学习的认识和理解。希望对大家有帮助

 1. 学习是认识世界——求是 - 学习是认识自己——求真

 学习的方法论

 1. 提出问题（学什么） - 要常实践（怎么学） - 要常总结（写下来） - 要有恒心
